## 🔥 Features

- ✅ **UI Moderna** - Interface elegante com animações suaves e design profissional
- 🎯 **ESP com Cores de Time** - Identifique prisioneiros, polícia e criminosos pelas cores
- 🎮 **Aimbot** - Mira automática com smooth ajustável e múltiplas verificações
- 🚀 **Fly System** - Voe livremente com velocidade ajustável
- ⚡ **Speed Hack** - Acelere seu personagem até 250 de WalkSpeed
- 🛡️ **Godmode** - Vida infinita
- 🔓 **Noclip** - Atravesse paredes e objetos
- 🦘 **Infinite Jump** - Pulos infinitos
- 📍 **TP Click** - Teleporte para onde você clicar
- 💤 **Anti-AFK** - Nunca seja desconectado por inatividade
- 🔒 **Anti-Stun** - Remove efeitos de stun e knockback
- 🔄 **Auto Rejoin** - Reconecta automaticamente ao sair
- 🚫 **Anti Kick** - Bloqueia kicks do servidor
- 🛡️ **Anti Crash** - Remove partículas e efeitos que causam lag
- ⚙️ **FPS Unlocker** - Desbloqueie FPS até 240
- 💡 **Fullbright** - Iluminação máxima no jogo
